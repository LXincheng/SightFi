## 4. Definition of Done

| 类别     | 标准                                             |
| -------- | ------------------------------------------------ |
| 代码质量 | 无阻断 lint/type 错误；无 `any`                  |
| 功能     | 对应任务验收项通过                               |
| 文档     | 至少更新 `goal/plan/architecture/rules/log` 之一 |
| 体验     | 桌面 + 移动关键路径可用                          |
| 安全     | Secret 不入前端包与日志                          |
| 弹性     | 外部依赖失败时链路可降级                         |
