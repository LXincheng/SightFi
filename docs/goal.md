# SightFi 项目目标（Goal）

> 更新时间：2026-03-03  
> 当前版本：MVP Phase 1

## 1. 北极星目标

| 维度     | 目标                                                         |
| -------- | ------------------------------------------------------------ |
| 产品价值 | 用“可追溯事实 + 实时资产状态 + 证据化 AI 输出”替代噪声信息流 |
| 用户体验 | 3 分钟内完成核心任务：看市场、看持仓、看事实新闻             |
| 工程原则 | 低耦合、可替换、可观测、可迭代                               |

## 2. MVP 交付边界

| 做什么                             | 不做什么               |
| ---------------------------------- | ---------------------- |
| 新闻事实流（来源、时间、关联标的） | 自动交易、策略代客执行 |
| 实时行情 + 组合估值                | 重型 BI 平台           |
| AI 组合总结（证据/置信度/风险）    | 绝对化投资建议         |
| 响应式 Web（桌面 + 移动）          | 复杂多端原生应用       |

## 3. 成功标准（MVP）

| 编号 | 指标           | 目标                                        |
| ---- | -------------- | ------------------------------------------- |
| G-01 | 关键链路可用性 | 月可用性 >= 99.9%                           |
| G-02 | 首屏体验       | Dashboard 首屏 TTI（P95）< 2.5s             |
| G-03 | 数据可信度     | 新闻条目 100% 包含 `source` + `publishedAt` |
| G-04 | AI 可解释性    | AI 输出 100% 包含证据与风险提示             |

## 4. 非功能约束

| 约束     | 要求                                 |
| -------- | ------------------------------------ |
| 安全     | Key 仅服务端可见；日志不输出敏感信息 |
| 可维护性 | 前后端分层清晰，禁止跨层直接耦合     |
| 可替换性 | 第三方数据源必须 adapter 化          |
| 国际化   | 所有用户文案必须走 i18n 字典         |
